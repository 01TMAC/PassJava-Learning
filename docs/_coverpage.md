# Java面试题管理系统

- PassJava是开源的面试题管理系统
- `后端`采用SpringBoot+SpringCloud搭建
- `B端`采用Vue-Admin，`C端`采用微信小程序
- 零碎时间利用小程序查看常见面试题，夯实Java基础
- 采用流行的技术，如 SpringBoot、MyBatis、Redis、 MySql、 MongoDB、 RabbitMQ、Elasticsearch，采用Docker容器化部署
